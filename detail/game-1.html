贪吃蛇

经典贪吃蛇游戏，使用原生JavaScript实现。控制蛇的移动方向，吃掉食物增长身体，避免撞到墙壁或自己的身体。

游戏预览

游戏说明

操作方式

使用键盘方向键或WASD键控制蛇的移动方向。

游戏目标

吃掉屏幕上出现的食物，每吃一个食物蛇身增长一节。

游戏结束

当蛇撞到墙壁或自己的身体时游戏结束。

计分系统

每吃掉一个食物得10分，得分越高游戏难度略有增加。

控制按键

↑ 或 W 向上移动
↓ 或 S 向下移动
← 或 A 向左移动
→ 或 D 向右移动

技术栈

HTML5 CSS3 JavaScript Canvas API
在新窗口中打开游戏
